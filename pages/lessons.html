classList

Свойство для работы с CSS-классами: add/remove/toggle для стилей без манипуляций строками.

add()

Метод classList.add('class') — добавляет класс к элементу, если его нет.

remove()

Метод classList.remove('class') — удаляет класс из элемента.

contains()

Метод classList.contains('class') — проверяет наличие класса (true/false).
